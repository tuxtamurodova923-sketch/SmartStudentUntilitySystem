# Smart Student Utility System
Console Application bo'lib, talabalar uchun kundalik hisob-kitob va rejalashtirish ishlarini osonlashtirish ishlarga yordam beradi.

## Asosiy kommandalar
1. Student Helper
2. Salary Calculator 
3. Password Tools

## 1.0 Student Helper
Ushbu bo'limda consoledan kiritilgan 5 ta bahoning o'rganadi.Eng yuqori bahoni topadi .Bahoni A, B, C, D, F darajaga ajratiladi.

1. O'rtacha bahoni hisoblash
2. Eng yuqori bahoni topish
3. Bahoni darajaga ajratish
4. Exit
